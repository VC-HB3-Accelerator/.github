# Программа акселерации

Мы открываем представительство в ИТ-хабе, разворачиваем песочницу и помогаем участникам настроить и протестировать платформу DLE

| Параметр | Значение |
|----------|----------|
| Формат | Онлайн-звонки, видеозаписи, чаты в приложении фонда |
| Срок программы и техсопровождения | 5 лет |
| Старт когорты | При наборе участников |
| Структура | 4 когорты × 70 групп по видам деятельности, от 25 до 50 участников в группе |

| Вы | Что получите в акселераторе | Подробнее |
|----|---------------------------|-----------|
| **Регулятор** | Инфраструктура для песочницы: блокчейн-реестр, сканер, смарт-контракты с местными идентификаторами | [for-regulators.md](for-regulators.md) |
| **Предприниматель** | Платформа DLE, настроенная под ваш бизнес, обучение, доступ к инвестициям | [for-entrepreneurs.md](for-entrepreneurs.md) |
| **Инвестор** | Поток проверенных компаний из акселератора, прозрачная отчётность через блокчейн | [for-investors.md](for-investors.md) |
| **Подрядчик** | Заказы на настройку платформы для участников, оплата из казны проекта | [for-contractors.md](for-contractors.md) |

## Как устроена работа

1. **Регулятор** выделяет грант → фонд открывает представительство и интегрирует местные идентификаторы в платформу
2. **Предприниматели** покупают лицензию → устанавливают DLE → регистрируются в группе на платформе по виду деятельности
3. **Подрядчики** выбирают задачи на платформе и получают оплату за их выполнение
4. **Инвесторы** анализируют метрики и выбирают компании для инвестиций

### Управление через голосование

Все ключевые решения принимаются держателями токенов через ончейн-голосование на платформе:

- Приоритеты разработки и обновлений
- Распределение средств казны
- Отбор компаний для инвестиций
- Изменение условий программы

1 токен = 1 голос. Кворум — 51%. Подробнее: [governance.md](governance.md)
